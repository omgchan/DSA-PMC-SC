## Project Description

This project is assigned as part of the **Data Structures course** for a student of **Patan Multiple Campus, BSc CSIT**.

The objective of this project is to understand and implement fundamental **data structures and algorithms** using the C programming language. It focuses on core concepts such as **searching, sorting, stack, queue, and linked list operations**.

---

## Topics Covered

### 1. Searching Algorithms

* Sequential (Linear) Search
* Binary Search

### 2. Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### 3. Stack

* Stack implementation using Array
* Stack implementation using Linked List
* Operations: PUSH, POP

### 4. Queue

* Linear Queue
* Circular Queue
* Queue implementation using Linked List
* Operations: INQUEUE, DEQUEUE

### 5. Linked List

* Singly Linked List
* Insertion and Deletion operations
